# Lorenz Verschingel

## Opdracht WebApps

[Link to site](https://lorenzverschingel.herokuapp.com/)

Runs on Heroku, so can be slow to connect.

### To run locally:
1. npm install
2. node server.js
3. start mongod
4. http://localhost:7070

### To test:
1. Make sure to have the initial database.
  1. node seed.js
2. gulp runTests
